## Project Overview:

The purpose of this project is to get comfortable using CSS Grid and Flex-Box. The goal is gonna be to create a testimonial section using a grid layout.

## Built With

- _CSS Grid_
- _Flex Box_

## Learning Reflection

This was a great project for me to get used to using CSS Grid. I was never really comfortable with grid before, but by reading the CSS Grid documentation provided by Front-end Mentor's responsive design learning path, as well as getting that hands-on experience, I was able to complete the Testimonials project
